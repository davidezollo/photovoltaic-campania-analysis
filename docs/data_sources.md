# Data Sources

The dataset used in this project was constructed by integrating several public datasets related to Italian municipalities and renewable energy infrastructure.

The data mainly comes from open and public sources, including:

- ISTAT – demographic and territorial statistics for Italian municipalities
- CORINE Land Cover – land use classification data
- Altaimpianti – photovoltaic installation data
- Other publicly available geospatial and territorial datasets

Raw datasets were downloaded from official or open-data portals and merged in Excel to build a municipal-level dataset for the Campania region.

The final dataset used in the analysis is available in the `data` folder of this repository.
